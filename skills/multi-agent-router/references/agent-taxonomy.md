# Agent Taxonomy

Use categories only after choosing the execution mode. Categories are for narrowing specialist choice, not for deciding whether delegation is worthwhile.

## Orchestration

Use when the problem is task splitting, dependency planning, context compression, or result integration.

- `agent-organizer`
- `context-manager`
- `knowledge-synthesizer`
- `multi-agent-coordinator`
- `task-distributor`
- `workflow-orchestrator`

## Discovery And Review

Use when the immediate need is evidence, mapping, bug isolation, review, or verification.

- `code-mapper`
- `search-specialist`
- `reviewer`
- `code-reviewer`
- `architect-reviewer`
- `debugger`
- `error-detective`
- `qa-expert`
- `test-automator`
- `performance-engineer`
- `browser-debugger`
- `accessibility-tester`

## Security And Compliance

Use when the task touches security posture, exploitability, auth, secrets, or regulated controls.

- `security-auditor`
- `security-engineer`
- `penetration-tester`
- `compliance-auditor`

## Frontend And UI

Use when the task is primarily user-facing or framework-specific on the client.

- `frontend-developer`
- `react-specialist`
- `nextjs-developer`
- `typescript-pro`
- `ui-designer`
- `ui-fixer`

## Backend And AI

Use when the task is service-side, integration-heavy, MCP-related, or model-backed.

- `backend-developer`
- `fullstack-developer`
- `api-designer`
- `mcp-developer`
- `ai-engineer`
- `llm-architect`
- `prompt-engineer`
- `python-pro`

## Data And Databases

Use when query behavior, schema, database safety, or database-specific performance is central.

- `database-administrator`
- `database-optimizer`
- `sql-pro`
- `postgres-pro`

## Infrastructure And Operations

Use when the task is about build, deployment, containers, platform design, or operational reliability.

- `build-engineer`
- `deployment-engineer`
- `devops-engineer`
- `docker-expert`
- `kubernetes-specialist`
- `platform-engineer`

## Docs And Guidance

Use when the main output is documentation, prompt behavior, or structured developer guidance.

- `documentation-engineer`
- `docs-researcher`
- `technical-writer`

## Generic Fallbacks

These are not repo catalog specialists. Use them only when delegation still helps and the curated routes are not strong enough.

- `explorer`
- `default`
- `worker`
